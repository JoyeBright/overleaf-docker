# Overleaf on Docker — Local Edition

Run **Overleaf Community Edition** locally with **Docker** and all LaTeX packages preinstalled — no setup needed.

---

## Guide
https://javad.pourmostafa.com/misc/local-overleaf-docker/

## Quick Start

```bash
git clone https://github.com/joyebright/overleaf-classicthesis-local.git
cd overleaf-classicthesis-local
./start-overleaf.sh
```

## Tested on
macOS M1 Sequoia
